MarkDown编写格式简述

发表于 2020-04-27 阅读次数：
本文字数： 351 阅读时长 ≈ 1 分钟

MarkDown编写方式：为了减少使用鼠标，专注敲键盘。

推荐软件：Typora。

以下所有顶前的关键词后空格下才能触发MarkDown格式，关键词中有结尾的的不需要

MarkDown语法简要规则

1.标题

#一级标题

##二级标题

#越多标题越小。

2.列表

无序关键字*，-

  • *1
  • *2
    • 1
    • 2

有序关键词1. 2. 3.

  1. 1.
  2. 2.

3.引用

*关键词 > *

变成这种效果。

多重>多重效果

4.图片与链接

图片关键词 ！【】（）

链接关键词 【】（）

小写输入格式。

5.粗体、划线与斜体

*粗体关键词 * **

斜体关键词 *

划线关键词 ~~

例如：粗体，斜体，划线。

6.代码框

*关键词 `` *

int main()

{

cout<<"Hellow world!"<<endl;

return 0;

}

7.分割线

关键词 ** * ，—
